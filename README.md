# 🚀 CodePilot AI – AI-Powered Code Debugging Assistant

CodePilot AI is a full-stack web application that helps developers debug, understand, and improve their code using AI. It analyzes code snippets, identifies errors, and provides clear explanations along with suggested fixes.

---

## 🧠 Features

- 🔍 **AI Code Debugging**
  - Detects syntax, logical, and runtime errors  
  - Provides clear explanations  

- 🛠️ **Fix Suggestions**
  - Suggests improvements and corrected code  

- 📖 **Code Explanation**
  - Explains code in simple, understandable terms  

- ⚡ **Fast API Response**
  - Backend built with FastAPI for efficient processing  

- 🎨 **Modern UI**
  - Clean, dark-themed interface for better developer experience  

---

## 🏗️ Tech Stack

### Frontend
- React (Vite)
- Tailwind CSS

### Backend
- Python
- FastAPI

### AI Integration
- Gemini API / OpenAI API

### Tools
- Git & GitHub

---

## ⚙️ How It Works

1. User inputs code or error  
2. Request is sent to backend API  
3. Backend processes input using AI  
4. AI returns:
   - Explanation  
   - Suggested fix  
   - Corrected code  
5. Results are displayed in a structured format
